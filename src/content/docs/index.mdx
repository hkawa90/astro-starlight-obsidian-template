# [Astro](https://astro.build/)の[Starlight](https://starlight.astro.build/) themeを試す

Obsidianで作成していたMarkdownをsiteで公開できるようにしてみた。

GithuのテンプレートとGithub Actionsを使ってローカルPCに導入しなくてもObsidianをWeb上に公開できます。

# Demo

今見えています。

サイドバーの`Notes`以降がObsidianから生成されたものです。

# Requirement

- Obsidianの`vaults`
- favicon.svg
  faviconファイルを`SVG`で用意しておきます。

# Configuration

設定自体はGithubの環境変数の設定で済みます。


import { FileTree } from '@astrojs/starlight/components';

下記の`public/favicon.svg`を用意してあったものと入れ変えます。またトップページ(今見ているページ)となる`src/content/docs/index.mdx`を変更します。

<FileTree>

- MyDoc/
- README.md
- astro.config.mjs
- package.json
- node_modules
- public/
  - favicon.svg
  - notes/
- src/
  - assets/
  - content/
    - docs/
      - index.mdx

</FileTree>
